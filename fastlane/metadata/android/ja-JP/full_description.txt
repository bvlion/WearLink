WearLink は、スマートフォンで登録した HTTP リクエストを Wear OS スマートウォッチと同期し、それぞれの端末から HTTP 通信を実行できるアプリです。スマートウォッチのタイルから、同期されたカスタムリクエストを瞬時に送信できます。

【WearLink の仕組みと主な特徴】

スマートフォンで HTTP リクエストを登録・管理：
送信先の URL、HTTP メソッド、必要なヘッダーやボディなどをスマートフォンアプリで設定・管理します。登録したリクエストは任意でスマートウォッチと同期できます。

Wear OS タイルから同期されたリクエストをワンタップ実行：
スマートフォンで登録し、スマートウォッチと同期された HTTP リクエストは、スマートウォッチのタイルからスマートフォンを取り出すことなく、手元で瞬時に実行できます。

実行履歴をスマートフォンで確認：
スマートフォンまたはスマートウォッチから実行した HTTP 通信の履歴は、スマートフォンアプリで確認できます。どのようなリクエストをいつ送信したかを把握するのに役立ちます。

こんな時に便利

IoT デバイスとの連携：
スマートウォッチから手軽に自宅のIoTデバイスにコマンドを送信（例：照明のON/OFF、家電の操作など）。スマートフォンが近くにない場合でも、スマートウォッチ単体で通信が可能です（Wi-Fi またはモバイルネットワーク接続が必要です）。

簡単な API の実行：
監視システムや個人用サーバーの簡単な API エンドポイントに、スマートフォンまたはスマートウォッチから素早くリクエストを送信して状態を確認。

特定情報の即時取得：
よく確認する Web API のエンドポイントを登録しておけば、スマートフォンまたはスマートウォッチからワンタップで情報を取得（例：天気情報、株価など）。

WearLink を活用して、スマートフォンとスマートウォッチそれぞれの端末から手軽に HTTP 通信を実行し、よりスマートな生活を実現しましょう！
